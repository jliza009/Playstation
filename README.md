# PlayStation Website

A modern, responsive PlayStation fan website showcasing top games, accessories, and news. Built with HTML, CSS, and JavaScript, the website is designed to offer an engaging experience for gamers.

- **Responsive Design:** Works smoothly on desktop, tablet, and mobile devices.

- **Accessories Showcase:** Displays essential PlayStation hardware including consoles, controllers, and VR.  
- **Other Games Section:** Showcases additional games such as Call of Duty, FIFA, and Rocket League.  
- **Contact Section:** Provides a clear way to reach out for inquiries.  
- **Interactive UI:** Hover effects and dynamic visual elements for an engaging user experience.
